# 数组

## 概念

数组是存储同一种数据类型的多个元素，长度不能改变的容器。



## 定义

`数据类型 [] 数组名; ` 推荐

`数据类型 数组名 []` C++风格



## 初始化

### 动态初始化

`数据类型 [] 数组名 = new 数据类型[数组长度] ;`

数组内元素拥有默认值

### 静态初始化

`数据类型 [] 数组名 = new 数据类型[]{元素1,元素2,...};`



## 常见异常

### ArrayIndexOfBoundsException

数组索引越界异常

### NullPointerException

空指针异常



# 二维数组

## 定义

`数据类型 [][] 数组名 ;`



## 初始化

`数据类型 [][] 数组名 = new 数据类型[长度][长度];`
